FuckItJS
========

### Javascript Boilerplate for Error (Message) Free Code

Installation
------------

1. Include "fuckit.js"
2. The End

API
---

**FuckIt**(*path_to_script*)

This will keep running your code until all errors have been sliced off like mold on a piece of perfectly good bread.
